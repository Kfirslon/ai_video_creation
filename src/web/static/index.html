Quick answers

Is this actually free?

Yes. Story + scene script run on free Groq + Gemini text APIs. Images generate via Gemini's free Nano Banana image API. The only manual step is dropping each scene's prompt into Google Flow (also free, with 1000 monthly credits). You can ship videos without a credit card.

Do I need to install anything?

Not on this hosted version — open the page, fill in the form, drop your generated mp4s back in. If you want to run it locally (faster, no quota worries), clone the repo from GitHub and double-click start.bat.

How long does one video take?

About 20 minutes of active time per video — most of which is waiting for Veo to render each clip. The script + 8 images take ~2 minutes total. Stitching with captions + audio mix is about 30 seconds.

Why do I have to paste prompts into Google Flow myself?

Google Flow has no public API. The Veo video model itself does have an API but it costs $0.10–$0.40 per second of generated video. To keep this tool free, you do the Flow paste step manually. We auto-generate the prompts, the images, the voiceover scripts, and the final stitch.

Can I post these to TikTok and Instagram?

Yes. The output is 1080×1920 H.264 MP4 with burned-in word-by-word captions — exactly the format Reels and TikTok prefer. Avoid using copyrighted background music unless you own the rights; Pixabay and the TikTok Commercial Music Library are good free sources.

Can I use the videos commercially?

The tool is MIT-licensed. The Gemini and Groq free tiers permit personal and commercial use within their rate limits. Always double-check the latest terms of any API or music source you bring in.

Are character looks consistent across scenes?

Mostly yes. Scene 1's image is passed as a reference when generating scenes 2–8, so the same character in the same outfit re-appears across the story. Occasionally Gemini drifts on small details — you can regenerate any single scene from the project folder if needed.
